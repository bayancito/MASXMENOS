# MASXMENOS

Plataforma informática de asistencia agrícola y comercio justo para pequeños productores rurales de Cochabamba, Bolivia.

## Descripción

MASXMENOS busca reducir la intermediación injusta en la comercialización agrícola mediante herramientas digitales accesibles, geolocalización y un marketplace orientado a pequeños productores.

## Tecnologías

### Backend
- Python
- Django

### Base de Datos
- PostgreSQL
- PostGIS

### Frontend
- HTML5
- CSS3
- JavaScript
- Bootstrap

### Geolocalización

- OpenStreetMap
